# Robo Top — Краснодар

Адаптивный лендинг реальной школы программирования и робототехники Robo Top для детей 4–15 лет в Краснодаре.

## Что реализовано

- авторский интерфейс без UI-фреймворков;
- фотографии из открытых публикаций школы;
- подтверждённые направления с фильтром по категории;
- адаптивное мобильное меню;
- блоки методики, проектов, преимуществ и FAQ;
- форма пробного урока с проверкой полей и демонстрационным успешным состоянием;
- адаптация для компьютеров, планшетов и телефонов;
- поддержка `prefers-reduced-motion` и базовая доступность.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```
